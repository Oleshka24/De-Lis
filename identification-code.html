Идентификационный налоговый номер ( ИНН)

Представляет собой индивидуальный налоговый номер, который занесен в Государственный реестр физических лиц и выдается только Государственной налоговой службой Украины.


Граждане Украины могут получить идентификационный код по месту своей прописки в районных налоговых администрациях, а граждане других стран могут получить его только в областной налоговой администрации.

Бюро переводов "De-lis" оказывает услугу по получению Идентификационного
кода для граждан других стран на основании доверенности.

Для чего нужен Идентификационный номер?

Идентификационный номер необходим для совершения ряда действий, а именно для покупки/продажи как движимого так и недвижимого имущества, для оформления доверенностей, для регистрации предприятий на территории Украины, для получения вида на жительство, для оформления гражданства и т.д.
